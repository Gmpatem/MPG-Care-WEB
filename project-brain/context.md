# Context — What This Project Is Trying to Achieve

## Business Purpose

MPG Technologies positions itself as a **business systems and automation partner** for companies that want to scale without manual work. The website's job is to:

1. Establish credibility through a premium visual design.
2. Explain the problem (manual work, scattered tools, missed bookings).
3. Present the solution (smart booking, automation, dashboards, custom dev).
4. Showcase a portfolio of engineered systems.
5. Convert visitors into **booked strategy calls** or **project inquiries**.

## Target Users / Clients

- **Primary:** Small-to-mid-sized service businesses (wellness studios, logistics, fintech, churches, etc.) that are still running on spreadsheets and manual processes.
- **Secondary:** Operations directors and CTOs looking to automate internal workflows.
- **Tertiary:** Non-technical teams that need no-code automation tools.

## Main Conversion Goals

1. **Book a free 30-minute strategy call** (highest priority).
2. **Submit a project inquiry / "Start a Project"** (medium priority).
3. **Browse portfolio and request a demo** (low priority, trust builder).

## Expected User Journey

1. **Awareness** → Lands on Hero, sees particle effect + headline.
2. **Problem recognition** → Scrolls to Value Proposition, identifies with pain points.
3. **Solution interest** → Views Services, sees "Learn more" (currently dead).
4. **Trust building** → Views Projects, Process, Testimonials, Trust metrics.
5. **Conversion** → Clicks "Book a Call" or "Start a Project" in FinalCTA.
6. **Success** → Calendar opens or form submits, confirmation/thank-you shown.

## Current Limitations Blocking the Journey

| Stage | Blocker | Impact |
|-------|---------|--------|
| Solution interest | "Learn more" on service cards does nothing | Users cannot explore services deeper |
| Trust building | Project cards have dead Live Demo / GitHub links | Portfolio feels fake |
| Conversion | "Book Your Free Call" button has no action | Zero calls booked |
| Conversion | "Start a Project" is `href="#"` | Zero inquiries submitted |
| Conversion | No dedicated `/book` or `/contact` page | No shareable or trackable conversion URLs |
| Success | No thank-you or confirmation state | Users get no feedback after attempting action |

## Honest Assessment

The site **looks professional** but **does not convert**. Every critical CTA either scrolls to another section (which also does not convert) or is a dead link. A visitor cannot actually book a call, start a project, or even send an email without manually copy-pasting the address from the footer.
